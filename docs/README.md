# Craft to Exile 2 가이드

Craft to Exile 2는 1.20.X Forge 기반 모드팩으로, 탐험과 커스터마이징에 중점을 둔 ARPG 스타일의 마인크래프트 경험을 제공합니다. Mine and Slash 모드를 핵심으로 하여 액션 RPG 스타일의 진행 시스템을 갖추고 있습니다.
