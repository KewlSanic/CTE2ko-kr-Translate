# VR 가이드

Craft to Exile 2를 VR로 플레이하는 방법을 안내합니다.

---

## 중요 경고사항

> **주의**: 게임 종료 전에 반드시 메인 메뉴에서 VR을 꺼야 합니다. 다음 로그인 시 다시 활성화하세요. 그렇지 않으면 게임 내 GUI가 표시되지 않을 수 있습니다.

---

## 시스템 요구사항

### 필수 소프트웨어
- **SteamVR** 설치 및 Steam 실행 필수
- **VR 헤드셋** (Oculus, Vive, Index 등)

### Java 설정
**Java 21** 사용 권장

1. [Oracle](https://www.oracle.com/java/technologies/downloads/)에서 Java 21 다운로드
2. 런처 설정에서 `javaw.exe` 경로 선택

### 권장 JVM 인자
```
-XX:+UseZGC -XX:+ZGenerational
```

### RAM 할당
- **권장**: 4-8GB
- 성능 문제 발생 시 8GB로 증가

---

## 첫 시작 설정

### 렌더링 설정
1. 첫 시작 시 **전체화면 모드**로 실행
2. 렌더링 거리를 낮게 설정
3. 안정화 후 점진적으로 설정 조정

### 셰이더 사용 시 권장 설정

| 설정 | 권장값 |
|------|--------|
| Anti-aliasing | 낮음 또는 끄기 |
| Shadows | 낮음 |
| Desktop Mirroring | 낮음 또는 끄기 |

---

## VR 활성화 방법

1. PC에서 게임을 활성화 상태로 유지 (컨트롤러가 작동하려면 필수)
2. 좌측 컨트롤러로 메뉴 열기
3. **Options** → **VR Settings** 이동
4. VR 모드 활성화

---

## 게임 내 VR 설정

### VR Settings 메뉴 항목

| 설정 | 설명 |
|------|------|
| **이동 방식** | 텔레포트, 자유 이동 등 선택 |
| **컨트롤러 설정** | 손 위치, 조작 방식 조정 |
| **라디얼 메뉴** | 빠른 아이템/스킬 접근 설정 |
| **화면 설정** | HUD 위치 및 크기 조정 |

### 이동 방식 옵션
- **텔레포트**: 멀미 감소, 포인트 앤 클릭 이동
- **자유 이동**: 조이스틱으로 자유롭게 이동 (멀미 주의)
- **룸스케일**: 실제 공간에서 움직임

---

## 컨트롤러 바인딩

### SteamVR에서 바인딩 편집

1. SteamVR 메뉴 열기
2. **Settings** → **Controllers** → **Manage Controller Bindings**
3. **Vivecraft** 선택
4. **Edit** 클릭하여 바인딩 수정

### 라디얼 메뉴 접근
- **Grip 버튼**을 눌러 대체 바인딩 세트에 접근
- 라디얼 메뉴로 빠른 아이템/스킬 전환 가능

### 기본 컨트롤러 조작

| 버튼 | 기능 |
|------|------|
| 트리거 | 공격/상호작용 |
| 그립 | 대체 동작/라디얼 메뉴 |
| 조이스틱 | 이동/회전 |
| 메뉴 버튼 | 인벤토리/일시정지 |

## 성능 최적화

### VR 전용 최적화 팁

1. **Render Distance 줄이기**: 8 청크 이하 권장
2. **셰이더 사용 주의**: 고사양 VR 셰이더는 성능에 큰 영향
3. **Reprojection 활성화**: SteamVR에서 Motion Smoothing 활성화
4. **백그라운드 앱 종료**: 시스템 리소스 확보

### 멀미 감소 팁

1. 텔레포트 이동 방식 사용
2. FOV 비네팅 활성화
3. 짧은 세션으로 시작하여 점진적으로 늘리기
4. 안정적인 프레임레이트 유지

## 일반적인 VR 문제

### GUI가 보이지 않음
- 게임 종료 전 VR 모드를 끄지 않아서 발생
- 해결: 메인 메뉴에서 VR 모드 껐다가 다시 켜기

### 컨트롤러가 작동하지 않음
- PC에서 게임 창이 활성화 상태인지 확인
- SteamVR 재시작

### 낮은 프레임레이트
- 그래픽 설정 낮추기
- RAM 할당 확인
- 백그라운드 프로그램 종료

### 추적 문제
- VR 플레이 공간의 조명 확인
- 베이스 스테이션/센서 위치 조정
