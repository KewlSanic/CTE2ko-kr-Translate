컨트롤러 및 스팀덱에서 플레이하는 방법

중요 제한사항

현재 컨트롤러로 3인칭 모드 플레이는 지원되지 않습니다. 1인칭 모드에서만 플레이해야 합니다.

컨트롤러 설정 (PC)

필요 모드

모드 버전 비고
Controller Support Mod 9.0.3 필수 설치
UnionLib 10.2.7 이미 포함됨

모드 설치 방법

  1. 모드 파일 다운로드
  2. \Craft to Exile 2\mods 폴더에 파일 복사
  3. 게임 재시작

컨트롤러 활성화

  1. Options → Controls → Controller Settings 이동
  2. 연결된 컨트롤러 선택
  3. 활성화 확인

키 바인딩 설정

  1. Options → Controls → Controller Binds 이동
  2. 변경하려는 버튼을 몇 초간 길게 누름
  3. 새로운 동작 할당

추가 설정

Options → Controls → Controller Extras에서 다음 설정 가능:

설정 설명
조이스틱 감도 이동/카메라 감도 조정
진동 컨트롤러 진동 설정
데드존 조이스틱 데드존 조정

스팀덱 설정

스팀덱에서 Craft to Exile 2를 플레이하려면 다음 단계를 따르세요.

설정 단계 (10단계)

  1. 데스크톱 모드 진입 - Steam 버튼 길게 누름 → 데스크톱 모드로 전환
  2. Prism Launcher 설치 - Discover 스토어에서 Prism Launcher 검색 및 설치
  3. 마인크래프트 계정 로그인 - Prism Launcher 실행 → Accounts → Add Microsoft 클릭
  4. 인스턴스 추가 - Add Instance 클릭 → CurseForge 탭 선택
  5. 모드팩 다운로드 - "Craft to Exile 2" 검색 → 다운로드 및 설치
  6. Steam에 Prism 등록 - Steam 열기 → Games → Add a Non-Steam Game → Prism Launcher 선택
  7. 데스크톱 모드에서 테스트 - Steam에서 Prism Launcher 실행 → 정상 작동 확인
  8. 게임 모드로 돌아가기 - Return to Gaming Mode 클릭 → Prism Launcher 실행
  9. Steam 키바인드 설정 - 컨트롤러 설정 버튼 클릭 → 키바인드를 WASD로 설정
  10. 키바인드 커스터마이징 - 필요에 따라 추가 키바인드 변경
내장 키바인드 관리자

스팀덱의 내장 키바인드 관리자를 사용하면 별도 모드 설치가 불필요합니다. Steam 오버레이에서 컨트롤러 설정에 접근하세요.

스팀덱 권장 설정

설정 권장값
Render Distance 6-8 청크
Graphics Fast
RAM 할당 4-6GB

배터리 수명 연장

  • 프레임 제한 설정 (30-40 FPS)
  • 셰이더 비활성화
  • 밝기 낮추기

일반적인 문제 해결

컨트롤러가 인식되지 않음

  • 컨트롤러 연결 확인
  • 다른 USB 포트 시도
  • 게임 재시작

버튼이 작동하지 않음

  • Controller Settings에서 컨트롤러 선택 확인
  • Controller Binds에서 바인딩 확인
  • 게임 내 Controls 메뉴에서 충돌 확인

조이스틱 드리프트

  • Controller Extras에서 데드존 값 증가

스팀덱 성능 문제

  • Render Distance 줄이기
  • 셰이더 끄기
  • 프레임 제한 설정
  • 백그라운드 앱 종료
